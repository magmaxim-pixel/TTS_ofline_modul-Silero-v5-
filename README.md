# Silero Offline TTS

Локальный синтезатор речи на базе моделей **Silero V5 (RU)** и **Silero V3 (EN)**.  
Работает полностью офлайн, поддерживает переключение языков, регулировку скорости/громкости и фоновое воспроизведение без открытия сторонних окон.

###  Особенности
*   ru **Русский язык**: Модель V5 (48 кГц), высокое качество, естественная интонация.
*   en **Английский язык**: Модель V3 (24 кГц), стабильный синтез.
*      **Быстродействие**: Оптимизации PyTorch 2.x ускоряют генерацию на 40–70%.
*      **Чистый звук**: Автоматическое добавление тишины в конец фразы убирает резкие обрывы.
*      **Настройки в реальном времени**: Меняй скорость (`speed`) и громкость (`vol`) прямо в диалоге.
*      **Модульная архитектура**: Код разделен на логические части (`core`, `audio`, `cli`) для легкой поддержки.

## Установка

1.  Убедись, что установлен **Python 3.9+**.
2.  Клонируй репозиторий или распакуй архив с проектом.
3.  Создай виртуальное окружение и установи зависимости:

```powershell
# Создание окружения
python -m venv venv
.\venv\Scripts\Activate

# Установка пакета в режиме разработки
pip install -e .
```

> **Примечание:** При первом запуске модели будут автоматически скачаны из репозитория Silero (требуется интернет). Далее работа идет полностью офлайн.

## Старт

Запусти интерактивный консольный интерфейс:

```powershell
python examples/main.py
```

Ты увидишь приглашение ввода, например: `[ru/xenia]`. Вводи текст на русском или английском, и система озвучит его.

## Команды управления

В интерактивном режиме доступны следующие команды:

| Команда | Описание | Пример |
| :--- | :--- | :--- |
| `lang <key>` | Сменить язык (`ru` или `en`) | `lang en` |
| `<speaker>` | Сменить голос (по имени, доступны спикеры русского) | `xenia`, `aidar` |
| `speakers` | Показать доступные голоса | `speakers` |
| `vol <0.1-2.0>` | Изменить громкость | `vol 1.2` (громче) |
| `speed <0.5-2.0>` | Изменить скорость речи | `speed 1.5` (быстрее) |
| `help` | Показать справку | `help` |
| `exit` / `q` | Выход из программы | `exit` |

### Доступные голоса
*   **🇷🇺 Russian (V5):** `xenia` (жен), `kseniya` (жен), `aidar` (муж), `baya` (жен).
*   **🇬🇧 English (V3):** `en_0` (универсальный).

## Использование в коде (API)

Ты можешь использовать `OfflineSileroTTS` как библиотеку в своих Python-скриптах.

### 1. Базовый синтез и сохранение в файл
```python
from silero_tts import OfflineSileroTTS

# Инициализация (загружает RU и EN модели)
tts = OfflineSileroTTS(languages=["ru", "en"], device="auto")

# Синтез русской фразы
tts.save_to_file("Привет! Это тест локального синтеза.", "output_ru.wav", speaker="xenia")

# Переключение на английский
tts.switch_language("en")
tts.save_to_file("Hello! This is offline synthesis.", "output_en.wav", speaker="en_0")
```

### 2. Получение аудио-массива (NumPy)
```python
import soundfile as sf

# Синтез возвращает numpy-массив
audio_data = tts.synthesize("Как дела?", speaker="aidar")

# Можно сохранить вручную или обработать
sf.write("custom_output.wav", audio_data, 48000)
```

### 3. Настройка устройства и оптимизаций
```python
# Принудительное использование CPU и отключение оптимизаций (для отладки)
tts = OfflineSileroTTS(
    languages=["ru"], 
    device="cpu", 
    optimize=False
)
```

## Конфигурация

Все настройки хранятся в `src/silero_tts/config.py`. Ты можешь изменить их под свои нужды:

*   `SILENCE_PADDING_SEC`: Длина тишины в конце фразы (по умолчанию `0.5` сек). Увеличь, если фразы обрываются слишком резко.
*   `DEFAULT_VOLUME`: Громкость по умолчанию (`0.8`).
*   `DEFAULT_SPEED`: Скорость речи по умолчанию (`1.0`).
